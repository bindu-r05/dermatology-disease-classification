# Dermatology Disease Classification

## Overview

This project focuses on the automated classification of dermatological diseases using Machine Learning techniques. Two approaches were developed and evaluated:

1. Multi-Layer Perceptron (MLP)
2. Radial Basis Function (RBF)

The system analyzes structured clinical data and classifies patients into six dermatological disease categories.

---

## Dataset

- Instances: 366
- Features: 34
- Type: Tabular Clinical Data
- Classes: 6

### Disease Categories

- Psoriasis
- Seborrheic Dermatitis
- Lichen Planus
- Pityriasis Rosea
- Chronic Dermatitis
- Pityriasis Rubra Pilaris

## Results

The developed models demonstrated strong classification performance on dermatological clinical data.

The RBF-based approach achieved approximately 97.30% classification accuracy.
